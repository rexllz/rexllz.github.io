Props, State, Render

当state和props发生改变时，render函数就会执行

render过程效率很高，因为有虚拟DOM

虚拟DOM

render的过程

  • State 数据
  • JSX 模板
  • 数据+模板 整合生成真实DOM
  • State发生改变
  • 数据+模板 整合生成真实DOM，替换原有DOM

多次生成DOM非常耗费性能

改良过程：

  • State 数据
  • JSX 模板
  • 数据+模板 整合生成真实DOM
  • State发生改变
  • 数据+模板 整合生成真实DOM，并不替换原有DOM
  • 对比差异
  • 局部替换

最终方案：虚拟DOM

  • State 数据
  • JSX 模板
  • 数据+模板 整合生成真实DOM
  • 生成虚拟DOM（一个js对象）
  • State发生改变
  • 生成新的虚拟DOM
  • 比较两个虚拟DOM的区别
  • 直接操作DOM，改变内容
